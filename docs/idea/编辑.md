| 快捷键                       | 介绍                  |
| ---------------------------- | --------------------- |
| `Shift+Ctrl+u`               | 大小写的切换          |
| `Shift + Alt + 鼠标左键点击` | 多处同时写            |
| `Alt + 鼠标左键拖动`         | 同时选中多行          |
| `Ctrl + w`                   | 选择光标所在的单词    |
| `Ctrl+D`                     | 复制一行              |
| `Ctrl+Y`                     | 删除一行              |
| `Ctrl+X`                     | 剪切一行              |
| `Ctrl+↑/↓`                 | 代码上/下滚动         |
