## 常用快捷键


1. **查看方法/变量的定义**

    `Ctrl + B` 进入光标所在的方法/变量的接口或是定义处，等效于 **Ctrl + 左键单击**。
    
    

2. **查看接口的实现**

     `Ctrl+Alt+b` 

​    

3.  **将当前行复制到下一行** 

    `Ctrl+D`

    

4. **跳转到下一个方法**

     `Alt + ↓`

     

5. **跳转到上一个位置**

    `Ctrl + Alt + ←/→`

    

6. **查看Java方法调用树(被调/主调)** 

     `Ctrl+Alt+h`

​    

7. **查看注释文档**

    实际项目中，通常一个类中的代码都不少，而且有很多的变量。那么如何快速知道这个变量的一些信息，比如类型，定义？

    有一个快捷键，可以查看方法或者变量的一些信息。第一步，移动光标至想要查看的变量或者方法上，第二步，`Ctrl + q`
